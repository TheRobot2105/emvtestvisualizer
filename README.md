# Instructions

- [Download](https://install.julialang.org/Julia.appinstaller) and Install Julia
- Download this folder
- Use VS Code (available in Company Portal) to open this folder
- Install the Julia Language extension in VS Code

    1. Add the CSV file you get from the GL240 to the `input` Folder
    2. Change the minimum and maximum Temperature of the plot in the top of `visualizer.jl`

    3. Press the play icon in the upper right corner and an interactive Plot should open after 20 to 30 seconds. In the folder `ouput` are now an SVG and HTML File.

- Using the Script for the first time can take a few minutes because the needed packages need to be downloaded and precompiled. This needs to be done only the first time and the script is faster the second time you use it.
